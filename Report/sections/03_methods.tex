In the figure, 10 harmonics are included in the x-direction, all with amplitude $10 \text{nm}$. This yields a total grating height of $200 \text{nm}$. The staircase is binarized by thresholding - if the sine wave in the given $z$ value is as high, it gets to be grating material, otherwise it is air. The curve is drawn on a x,y resolution, far above the Nyquist limit of the highest grating profile order (here $n_x$ = 5000 corresponding to a frequency of $5\frac{\text{pixels}}{nm}$, while the highest grating order is has a frequency of $0.01\frac{\text{oscillations}}{nm}$).
The hars edges come solely from the discretization along the z-direction. Since every layer is fourier transformed, step-functions edges require more fourier orders to resolve. This is a fundamental theorem for the convergence of Fourier series on discontinuous functions. Also, note the third layer in figure \ref{fig:staircase}  (a). Here, the grating is parameterized as a gradient index material of constant height, but oscillating refractive index. Though both of these cases are simplifications and have numerical artifacts, the convergence tests below show that the smoothing improves convergence. The smoothing is done by simply dividing the height of the grating oscillation that is within the given layer height. The code snippet is shown below.
\begin{lstlisting}
    def get_staircase_sine_eps(x, params_x, grating_period, num_layers, eps_high, eps_low=1., subpixel=True, y=None, params_y=None, grating_period_y=None):
    """Generate sine grating in staircase approximation. Supports 1D and 2D gratings."""
    params_x, params_y_split = split_params_3d(params_x)
    params_y = params_y if params_y is not None else params_y_split
    grating_period_y = grating_period_y or grating_period
    
    profile_x, height_x = _compute_1d_profile(x, params_x, grating_period)
    # Here was logic for the 3d profile if params_y is given. It is a sum of two _compute_1d_profiles.

        eps_profile = profile_x
        grating_height = height_x
        nx = x.shape[0]
        thresholds = torch.arange(0, num_layers, device=x.device) * (grating_height / num_layers)
        thresholds = thresholds.unsqueeze(0).expand(nx, -1)
        eps_profile = eps_profile.unsqueeze(1).expand(-1, num_layers)

    cell_height = grating_height / num_layers
    eps = torch.clamp((eps_profile - thresholds) / cell_height, min=0, max=1)
    
    if not subpixel:
        # Straight-Through Estimator trick
        # This avoids killing gradients
        eps = (torch.round(eps) - eps).detach() + eps
        
    return eps_low + (eps_high - eps_low) * eps
\end{lstlisting}
The machine learning target of this project is the absorptance in the film layer. As discussed previously, I found the dissipated power in the film layer by Poynting analysis. For the incident power, I ran a dummy simulation with just a free space input and output layer, with the same incident conditions of light. I kept track of the absorptance in both polarizations, which thanks to the $S$-matrix formulation of RCWA was obtained simultaneously. In COMSOL, a simulation for each polarization was made.
The rest of the RCWA imlementation was about using TORCWA, and I will therefore just refer again to their article \cite{torcwa}.

\section{COMSOL implementation of a grating}
\label{sec:comsol}
A fourier patterned grating of a single oscillation with no phase shift and an amplitude of $50\text{nm}$ was implemented in COMSOL. Using COMSOL's built in material library, which refers to \url{https://refractiveindex.info/}, the geometry was split into multiple different domains.
Figure \ref{fig:comsol} shows three images. Image (a) is an image of the geometry. The top domain, over the oscillation is silicon. Light propagates in the z direction, as shown by the small red arrow. The first domain under the oscillation is air. The boundary on the side are periodic, and the boundary at the bottom is not a perfect electric conductor, but an impedance boundary condition with refractive index $n = 0 + 100i$. This is done because RCWA doesnt have direct boundary conditions in the layer S-matrices. Or that is, as discussed in section \ref{sec:rcwa}, the standard boundary conditions of Maxwell's equations determine the S-matrices.
As such, I have approximated a perfect electrical conductor (mirror) by an artificially high complex refractive index. This will have a small amount of phase change, but it is almost negligible. The bottom domain, the rectangle, is a perfectly matches layer - here reflected electrical fields disappear, so they dont scatter back and interfere.
(b) shows the mesh, made according to the rule of thumb that there should be a minimum mesh size of at most $s_{mesh} = \frac{\lambda}{5n(\lambda)}$. To be sure that the simulation is high fidelity I used an even finer mesh size, closer to $s_{mesh} = \frac{\lambda}{60n(\lambda)}$ at the boundary between materials, and $s_{mesh} = \frac{\lambda}{10n(\lambda)}$ everywhere else. Figure (c) shows a close up of the fine triangular mesh.

\section{Convergence tests}
To ensure the numerical stability and accuracy of the RCWA solver, rigorous convergence testing is performed. All of the plots included here can be found for equivalent conditions except without the smoothing described in the last section in the appendix \ref{sec:nosubpix}. They can also be found for a grating of a height that is 10 times higher. The conclusions are much the same, but they are included for completeness sake.
Convergence criteria largely depend on the number of Fourier orders $N$ retained and the number of layers in the staircase approximation of the fourier patterned grating.
In both cases of figure \ref{fig:convergence_num_layers} and \ref{fig:convergence_order_N}, the error is compared relative to the absorptance curved obtained from the high fidelity comsol simulation of a single fourier patterned grating.
The incident angle for these convergence simulations is $30\deg$, for no other reason than having some field components in all directions. Especially of note is it that the p-polarization has a field component in the propagation direction $z$.
as already discussed in section \ref{sec:rcwa}, discontinuities at boundary conditions, especially for the propagation direction. This theoretical expectation fits with the observed behavior of the convergence tests for RCWA vs increased number of layers and increased order of Fourier truncation. The sudden stop in improvement for figure \ref{fig:convergence_order_N} at 10 Fourier orders for s-polarization only suggest that here, the staircase approximation is fully resolved in diffraction orders.

Figure \ref{fig:convergence_order_N} shows convergence vs truncated Fourier order at a constant number of layers. Both figures simulates the same grayscale structure,
a Fourier patterned grating of the form of equation \ref{eq:grating} with only the first amplitude nonzero at $a_x=50$, meaning the grating has a total height of $100$ nm.
A general rule of thumb for convergence tests is that the overall physics is captured, when the improvement of the parameter under question is linear in a log plot. It might be argued that this occurs already at 10 layers yielding $10 \text{nm}$ per layer for the grating staircase,  and a fourier truncation order of 10. This also suggests that the variable held constant is sufficient for each plot.
For these values, the mean error is below 0.01, or since it is average absorptance, correspond to an error of $\pm 1\%$ of absorbed light power. The max error for these values is on the order of $0.1$ or $10\%$ absorptance error. Though this sounds substantial, it is most likely a poorly resolved resonance peak as discussed in section \ref{sec:fabry-perot}, and the finite small width makes the weight of the error negligible. The average error is more important.
In addition to just the errors, the actual spectra are plotted for a selection of different number of layers in the appendix, section \ref{sec:comsolSpectra}.

With this it has been shown that RCWA matches COMSOL for a high enough fourier truncation order and height per layer in the sinusoidal grating layer.
As such, convergence tests can now be made against RCWA itself for a very high scattering order and number of layers. \cite{HOPE} argues that sinusoidal modulations scatter incoming electromagnetic fields to the order of the sine modulation of the material.
To see what happens for a low truncation order, a convergence test against the sinc like grating plotted in figure \ref{fig:staircase}, with 10 oscillations of amplitude $10\text{nm}$ each has been made, and is shown in figure \ref{fig:convergence_combinations}. The right side plot has the same sinc like-structure, but with the oscillations in both the $x$- and $y$ directions.
The truncated fourier order of the 3D test is the same for $x$ and $y$. This convergence test is especially interesting for 3D, and it is interesting to see whether the number of included harmonics in the grayscale Fourier patterened grating layer has to follow the Nyquist limit of twice as many diffracted RCWA orders. For the 2D combinations at the left, the RCWA order is taken all the way to $100$. The difference in time complexity of truncated order 100 in 2D and 3D is $10^6$, so the 3D order is capped at $12$.
For the 2D combinations to the left in the figure, mean absolute error is close to $10^-3$ already for 10 truncated orders and $10$ staircase layers. Since the grayscale modulated grating contains 10 harmonics, this suggests that optical performance is not Nyquist limited by the truncated fourier order of RCWA.
For the 3D combinations, very good performance at an error of less than $1\%$ already at 3 truncated orders is observed. As discussed in section \ref{sec:rcwa}, for 3D periodic gratings, there really exists $~N^2=9$ diffracted orders, which here corresponds to about an order of $10$, like the 2D results.

The $1\%$ mean error discussed in last section represents in some way the aleatoric, or irreducible, uncertainty of the final dataset\footnote{\url{https://en.wikipedia.org/wiki/Uncertainty_quantification}}. Even with more datapoints, this error stems from the choice of solver and error from finite methods. Apart from making the problem exponentially more complex, there is nothing to do to reduce it, and it would still only get reduced linearly in a log scale.
The final training loss/validation error of the machine learning surrogates represents in some way the epistemic uncertainty  of the surrogate models. This could be reduced by tracking the loss differently, improving model expressiveness, matching the number of model parameters to the problem at hand, or any number of improvements on the model side. But the final frontier can never be better than the $1\%$ error of the dataset itself.

During training of the machine learning surrogate models, weights were stored using exponential moving average. The moving average was handled by a small custom tracker, seen below
After every single batch, the weights of the tracker are updated smoothly by exponential decay, as seen below in the code snippet.
\begin{lstlisting}
    #...code cut out above...
    "self._shadow stores the weights, self._best_snapshot stores the best seen weights"
    @torch.no_grad()
    def update(self, model: nn.Module) -> None:
        """Step the EMA after one optimiser update."""
        d = self.decay
        for name, param in model.named_parameters():
            self._shadow[name].mul_(d).add_(param.data, alpha=1.0 - d)

    def snapshot(self) -> None:
        """Save the current EMA shadow as the best seen so far."""
        self._best_snapshot = {k: v.clone() for k, v in self._shadow.items()}
\end{lstlisting}
