# Antigravity CLI Rules & Memory

1. **Artifact Archiving**: Before starting a completely new plan or significantly pivoting tasks, always archive the existing `implementation_plan.md`, `task.md`, and `walkthrough.md` artifacts by renaming them with a descriptive prefix (e.g., `3D_refactor_plan.md` or `hpc_submit_scripts_plan.md`). Do not blindly overwrite them!
2. **Project Context**: This project involves simulating light trapping layers using RCWA (TORCWA). The HPC cluster is DTU LSF.
